# FeaChat
FeaChat is the first chat app I developed using Python.

## /server/

### Initialization

First-time:
```
python main.py --init
```
Afterwards:
```
 python main.py
 ```

## /client/

Switch in core.py
```
DEV_MODE = True/False
```
